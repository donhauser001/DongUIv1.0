# 项目结构说明

本文档介绍了 **Dong-FullStack** 项目的目录结构约定。

## 目录树

```text
apps/web/src/
├── api/             # API 接口定义
├── config/          # 全局配置 (主题默认值)
├── components/      # 公共组件
├── layouts/         # 布局组件 (AdminLayout, UserLayout)
├── router/          # 路由配置
├── docs/            # ★ 文档中心数据源
├── views/           # 页面视图
│   ├── admin/       # 后台页面 (含组件画廊 ComponentGallery)
│   └── user/        # 前台页面
├── utils/           # 工具函数 (theme.ts 等)
├── style.css        # 全局样式与 CSS 变量定义
└── main.ts          # 入口文件
```

## 约定

1.  **视图文件**: 所有页面级组件放在 `views` 目录下。
2.  **组件命名**: 始终使用 PascalCase (如 `UserDetail.vue`)。
3.  **主题开发**:
    *   `views/admin/dev/ComponentGallery.vue`: **组件画廊**，即项目的核心主题设置与预览中心。
    *   `config/theme.ts`: 定义默认的主题配置对象。
    *   `style.css`: 定义全局 CSS 变量和基础语义类 (`.btn-primary`, `.card-base` 等)。
