# DongUI 设计规范 v1.0

本文档详细阐述了 **DongUI** 的视觉语言和开发规范。本系统基于 **原生 CSS 变量 (CSS Variables)** 和 **语义化 CSS 类** 构建，实现高度统一且可完全配置的 UI 风格。

## 1. 设计哲学 (Design Philosophy)

### 1.1 核心理念
- **专业与冷静**: 默认采用 **孟加拉绿 (Bangladesh Green)** 作为主色调，传达学习、成长与冷静的心理暗示。通过 CSS 变量系统，主色调可全局一键替换。
- **精致的线性感**: 摒弃厚重的阴影和复杂的渐变，全面采用 **1px 实线边框**。这种设计让界面显得轻盈、锐利且信息密度高。
- **内容为王**: 严格控制装饰元素，利用色彩变体（Primary/White/Dark）来区分信息的层级，而非依赖复杂的布局。

---

## 2. 色彩系统 (Color System)

所有颜色均通过全局 CSS 变量管理，请通过语义化类名或 CSS 变量使用，**严禁硬编码 Hex 值**。

### 2.1 主色系 (Primary)
用于导航栏高亮、主要按钮、激活状态。

| 语义变量 | 默认值示例 | CSS 类示例 | 用途 |
|---|---|---|---|
| **`--color-primary`** | `#03624C` | `.text-primary` / `.bg-primary` (需自定义) | 核心品牌色，用于高频交互点 |
| **`--color-secondary`** | `#00cf3f` | - | 辅助色，用于次级强调 |
| **`--color-accent`** | `#26d849` | - | 装饰色，用于图表或图标 |

### 2.2 功能色 (Functional)
用于表达状态。

| 状态 | 变量名 | 默认色 | CSS 样式示例 |
|---|---|---|---|
| **Success** | `--color-success` | Green 500 | `color: var(--color-success)` |
| **Warning** | `--color-warning` | Yellow 500 | `color: var(--color-warning)` |
| **Error** | `--color-error` | Red 500 | `color: var(--color-error)` |
| **Info** | `--color-info` | Blue 600 | `color: var(--color-info)` |

### 2.3 中性色 (Neutral)
用于文本、背景和边框。

| 元素 | 变量名 | 说明 |
|---|---|---|
| **主要标题** | `--color-text-primary` | 页头、卡片标题 |
| **正文内容** | `--color-text-primary` | 表单文字、文章 |
| **辅助文字** | `--color-text-secondary` | 日期、备注、标签 |
| **占位符** | `--color-text-tertiary` | Placeholder |
| **通用边框** | `--color-border` | **#211f20** (默认) |

## 3. 核心组件规范 (Component Specs)

### 3.1 按钮 (Buttons)
按钮样式完全由 `--btn-*` 系列变量控制。

- **主要按钮**: `class="btn-primary"`
  - 实心背景 (`--btn-primary-bg-color`)，白色文字。用于“提交”、“保存”、“新建”。
- **次要按钮**: `class="btn-outline"`
  - 白底，边框 (`--btn-outline-border-color`)，文字变色。用于“取消”、“导出”、“查看”。

### 3.2 卡片 (Cards)
系统中的卡片分为三个视觉层级：

1.  **标准卡片** (`class="card-base"`)
    - 样式由 `--card-*` 变量控制（内边距、圆角、边框、阴影等）。
2.  **强调卡片**
    - 在标准卡片基础上，增加主色边框或背景强调。
3.  **深色卡片**
    - 使用 `--color-dark` 作为背景，白色文字。

### 3.3 表格 (Tables)
表格采用**线性设计**风格，使用 `class="table-base"`：
- **可控边框**: 边框宽度和颜色由变量控制。
- **条纹/悬停**: 支持斑马纹配置 (`--table-zebra-stripe`) 和悬停高亮 (`--table-hover-row-bg-color`)。
- **表头**: 浅色背景 (`--table-header-bg-color`)。

### 3.4 输入框 (Inputs)
使用 `class="input-base"`：
- **统一高度**: `--input-height`。
- **聚焦态**: 聚焦时显示主色边框 (`--input-focus-border-color`) 和光晕 (`--input-focus-ring-color`)。

---

## 4. 布局与间距 (Layout & Spacing)

### 4.1 间距约定
不再依赖 Utility Classes 的固定网格，而是使用 CSS 变量或标准 REM 单位。

- **布局变量**: `--layout-content-padding` 等变量控制主要区域边距。
- **组件间距**: 推荐使用 CSS `gap` 属性配合 Flexbox/Grid 布局。

### 4.2 字体
- **中文字体**: 使用 `--font-family` 变量定义的系统无衬线字体栈。
- **大小**: 使用 `--font-size-*` 系列变量 (base, sm, lg, xl, 2xl)。
- **字重**: 使用 `--font-weight-*` 系列变量 (normal, medium, bold)。

## 5. 开发速查表 (Cheatsheet)

| 想实现... | 用这个类名 / 变量 |
|---|---|
| **核心组件** | `.btn-primary`, `.btn-outline`, `.input-base`, `.card-base`, `.table-base` |
| **引用主色** | `var(--color-primary)` |
| **标准圆角** | `var(--radius)` |
| **标准边框** | `border: var(--border-width) solid var(--color-border)` |
| **Flex 居中** | CSS: `display: flex; align-items: center; justify-content: center;` |

> **记住**: 保持简单。优先复用 `.card-base` 等语义化类名，而不是手动编写大量样式。
