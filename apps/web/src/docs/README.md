## 🚀 快速开始

### 安装依赖

```bash
pnpm install
```

### 启动开发服务器

```bash
# 启动前端和后端
pnpm -r run dev
```

### 访问应用

- 前台门户：http://localhost:5173
- 后台管理：http://localhost:5173/admin
- 后端 API：http://localhost:50000

## 🎨 核心特性

### 主题系统

- ✅ 全局主题配置（颜色、字体、布局）
- ✅ 实时预览和调整
- ✅ 组件画廊（可视化配置所有组件样式）
- ✅ 支持导入/导出主题配置

### 组件库

- ✅ 富文本编辑器（支持图片上传、颜色选择、格式化）
- ✅ 统一的表单组件（按钮、输入框、下拉框、开关等）
- ✅ 卡片、标签、模态窗、提示组件
- ✅ 响应式布局系统

### 用户管理

- ✅ 用户列表和详情
- ✅ 角色管理（RBAC）
- ✅ 权限控制

### 开发工具

- ✅ 组件画廊 - 可视化配置和预览所有组件
- ✅ 开发文档 - 完整的开发指南
- ✅ 主题编辑器 - 实时调整全局样式

## 📦 技术栈

### 前端

- **框架**: Vue 3 + TypeScript
- **构建工具**: Vite
- **路由**: Vue Router
- **样式**: 原生 CSS + CSS Variables
- **图标**: UnoCSS Icons (Carbon Design)

### 后端

- **框架**: NestJS + TypeScript
- **数据库**: PostgreSQL + Prisma ORM
- **API**: RESTful API

## 🔧 开发建议

1. **遵循规范** - 查看 [Git 规范](./01_规范指南/02_Git规范.md) 和 [UI 设计规范](./01_规范指南/03_UI设计规范.md)
2. **使用主题变量** - 所有颜色和样式使用 CSS 变量，参考 [主题变量使用指南](./02_组件使用/02_主题变量使用指南.md)
3. **组件复用** - 优先使用现有组件，避免重复造轮子
4. **文档先行** - 新增功能时同步更新文档
