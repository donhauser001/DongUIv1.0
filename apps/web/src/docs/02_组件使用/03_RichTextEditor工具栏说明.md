## 🎯 使用场景

### 写文章
1. 点击 **H1** 输入文章标题
2. 点击 **正文** 开始写内容
3. 使用 **粗体**、**斜体** 强调重点
4. 用 **列表** 组织要点

### 排版美化
1. 点击 **居中** 让标题居中
2. 点击 **标准** 设置舒适的行距
3. 用 **高亮** 标记重要内容
4. 用 **分隔** 分隔不同章节

### 插入内容
1. 点击 **图片** 上传本地图片
2. 点击 **链接** 添加网页链接
3. 用 **编号** 创建步骤列表

## 💡 使用技巧

### 1. 标题层级
```
H1: 文章大标题（每篇只用一次）
H2: 主要章节标题
H3: 子章节标题
正文: 普通段落内容
```

### 2. 行距选择
```
紧凑 (1.0): 代码、表格
标准 (1.5): 正文阅读（推荐）
宽松 (2.0): 需要批注的文档
```

### 3. 列表使用
```
列表 (•): 无顺序的要点
编号 (1.): 有顺序的步骤
缩进/减少: 调整列表层级
```

### 4. 颜色使用
```
颜色: 改变文字颜色（谨慎使用）
高亮: 标记重点内容（黄色背景）
```

## 🚫 常见问题

### Q: 为什么有些按钮看起来一样？
A: 相同功能的按钮（如H1/H2/H3）会用文字区分，鼠标悬停可以看到详细说明。

### Q: 如何取消格式？
A: 选中文字后点击 **清除** 按钮，可以移除所有格式。

### Q: 图片上传失败怎么办？
A: 检查图片大小是否超过5MB，格式是否为常见图片格式（jpg/png/gif）。

### Q: 快捷键不生效？
A: 快捷键只在编辑器聚焦时有效，点击编辑区域后再使用。

## 🎨 自定义工具栏

如果你觉得工具栏太复杂，可以自定义：

```vue
<script setup lang="ts">
// 简化版工具栏（只保留常用功能）
const simpleToolbar = [
  { command: 'bold', icon: 'i-carbon-text-bold', title: '粗体', tooltip: '粗体' },
  { command: 'italic', icon: 'i-carbon-text-italic', title: '斜体', tooltip: '斜体' },
  { command: 'separator', icon: '', title: '' },
  { command: 'formatBlock', icon: '', title: 'H2', tooltip: '标题', value: 'h2' },
  { command: 'formatBlock', icon: '', title: '正文', tooltip: '正文', value: 'p' },
  { command: 'separator', icon: '', title: '' },
  { command: 'insertUnorderedList', icon: 'i-carbon-list', title: '列表', tooltip: '列表' },
  { command: 'insertImage', icon: 'i-carbon-image', title: '图片', tooltip: '图片' },
]
</script>

<template>
  <RichTextEditor
    v-model="content"
    :toolbar="simpleToolbar"
  />
</template>
```
