```vue
  <label class="form-label">背景颜色</label>

    <input v-model="config.button.primary.backgroundColor" type="color" class="color-preview" />

  </div>
</div>
```

## 🎯 效果说明

现在每个颜色设置项都包含：

1. **圆形颜色预览按钮**（左侧）
   - 宽高：2.5rem × 2.5rem
   - 圆形显示
   - 点击打开系统拾色器
   - 显示当前颜色

2. **文本输入框**（右侧）
   - 显示颜色值（如 `#03624C` 或 `var(--color-primary)`）
   - 可手动输入颜色值
   - 支持 HEX、RGB、CSS 变量等格式
   - 与颜色预览按钮同步

## 🔧 技术细节

### 使用的 CSS 类：
- `.form-group`：表单组容器
- `.form-color-row`：颜色行容器（flex 布局）
- `.color-preview`：圆形颜色预览按钮
- `.form-input`：标准输入框样式
- `.color-text`：文本输入框（flex: 1）

### 样式定义位置：
所有样式已在 `apps/web/src/views/admin/dev/gallery.css` 中定义。
