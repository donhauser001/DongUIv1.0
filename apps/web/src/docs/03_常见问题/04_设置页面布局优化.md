# 设置页面布局优化

## 问题描述
设置页面的标题和设置项之间的间距不合理：
- 标题离上面的设置项太近
- 标题离下面自己的设置项太远
- 视觉层次不清晰

## 解决方案

### 1. 新增 CSS 类

在 `gallery.css` 中添加：

```css
/* Settings Item - 单个设置项的容器 */
.settings-item {
    display: flex;
    flex-direction: column;
    gap: 1rem;
}

/* Section Title - 增强视觉层次 */
.section-title {
    font-size: 1rem;
    font-weight: 600;
    color: #111827;
    margin: 2rem 0 1rem 0;
    padding-bottom: 0.5rem;
    border-bottom: 2px solid var(--color-primary);
    display: inline-block;
}

.section-title:first-child {
    margin-top: 0;
}

/* Settings Group - 设置组之间的间距 */
.settings-group {
    display: flex;
    flex-direction: column;
    gap: 2.5rem;
}
```

### 2. HTML 结构调整

**错误的结构**（标题和内容分离）：
```vue
<div class="settings-group">
  <h5 class="section-title">尺寸与形状</h5>
  <div class="grid-2">
    <!-- 设置项 -->
  </div>

  <h5 class="section-title">正常状态</h5>
  <div class="grid-2">
    <!-- 设置项 -->
  </div>
</div>
```

**正确的结构**（标题和内容作为一个整体）：
```vue
<div class="settings-group">
  <!-- 尺寸与形状 -->
  <div class="settings-item">
    <h5 class="section-title">尺寸与形状</h5>
    <div class="grid-2">
      <!-- 设置项 -->
    </div>
  </div>

  <!-- 正常状态 -->
  <div class="settings-item">
    <h5 class="section-title">正常状态</h5>
    <div class="grid-2">
      <!-- 设置项 -->
    </div>
  </div>
</div>
```

### 3. 视觉效果

- ✅ 标题与其对应的设置项紧密相连（gap: 1rem）
- ✅ 不同设置组之间有明显间隔（gap: 2.5rem）
- ✅ 标题有下划线强调，视觉层次清晰
- ✅ 第一个标题没有上边距，避免顶部空白过多

### 4. 需要修改的文件

所有设置组件都需要按此结构调整：
- ButtonInputSettings.vue
- FormSettings.vue
- CardSettings.vue
- TagSettings.vue
- ModalSettings.vue
- NotificationSettings.vue
