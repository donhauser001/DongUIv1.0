# 颜色输入框样式统一指南

## 🎯 问题描述

组件设置页面中的颜色输入框样式需要与全局配色页面保持一致。

## ✅ 正确的样式格式

### 全局配色页面使用的格式：

```vue
<div class="form-group" style="gap: 0.5rem;">
  <label class="form-label">背景颜色</label>
  <div class="form-color-row">
    <input v-model="config.colors.primary" type="color" class="color-preview" />
    <input v-model="config.colors.primary" type="text" class="form-input color-text" />
  </div>
</div>
```

### 关键点：
1. **外层容器**：`class="form-group" style="gap: 0.5rem;"`
2. **颜色行容器**：`class="form-color-row"`
3. **颜色选择器**：`type="color" class="color-preview"`
4. **文本输入框**：`type="text" class="form-input color-text"`
5. **两个输入框绑定同一个变量**

## ❌ 错误的格式

```vue
<!-- 错误：直接使用 form-input 类 -->
<div class="form-group">
  <label class="form-label">背景颜色</label>
  <input v-model="config.button.primary.backgroundColor" type="color" class="form-input" />
</div>
```

## 🔧 批量替换模式

### 查找模式：
```vue
<div class="form-group">
  <label class="form-label">XXX颜色</label>
  <input v-model="config.xxx.xxx" type="color" class="form-input" />
</div>
```

### 替换为：
```vue
<div class="form-group" style="gap: 0.5rem;">
  <label class="form-label">XXX颜色</label>
  <div class="form-color-row">
    <input v-model="config.xxx.xxx" type="color" class="color-preview" />
    <input v-model="config.xxx.xxx" type="text" class="form-input color-text" />
  </div>
</div>
```

## 📊 预期效果

修改后，每个颜色设置项将显示为：
- 左侧：圆形颜色预览按钮（点击打开拾色器）
- 右侧：文本输入框（显示和编辑颜色值，如 #03624C 或 var(--color-primary)）

这与全局配色页面的样式完全一致。

## ⚠️ 注意事项

1. **不要遗漏 `style="gap: 0.5rem;"`**：这确保标签和输入框之间的间距正确
2. **两个输入框绑定相同变量**：颜色选择器和文本框需要同步
3. **保持 `.form-input` 类**：文本输入框仍然需要这个类来保持样式一致
4. **添加 `.color-text` 类**：确保文本框占据剩余空间
